    Configurazioni Consigliate

    Quattro proposte pronte all'uso — una per ogni piano HyperBeauty. Ogni configurazione abbina il piano software al PC Windows e alla stampante telematica Custom ottimali per quella fascia di centro. Scegli quella del tuo cliente, verifica la disponibilità con il tuo Area Manager e compila il preventivo.

    Processore Intel® Celeron Bay Trail J1900
    Memoria 4 GB RAM
    Storage 64 GB SSD
    Display 15" TFT LED — formato 4:3 — luminosità 350-400 cd/m²
    Risoluzione 1024 × 768
    Touch True-Flat Projected Capacitive
    OS supportati Windows 7, Windows 10, Linux
